cascade config
==============

Synopsis
--------

:program:`cascade config` ``[OPTIONS]`` ``<COMMAND>``

Description
-----------

Manage Cascade's configuration.

Options
-------

.. option:: -h, --help

   Print the help text (short summary with ``-h``, long help with ``--help``).

Commands
--------

.. subcmd:: reload

   Reload Cascade's configuration
