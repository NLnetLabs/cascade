cascade template
================

Synopsis
--------

:program:`cascade template` ``[OPTIONS]`` ``<COMMAND>``

Description
-----------

Print example config or policy files.

Options
-------

.. option:: -h, --help

   Print the help text (short summary with ``-h``, long help with ``--help``).

Commands
--------

.. subcmd:: config

   Generate a config template

.. subcmd:: policy

   Generate a policy template
